== Description ==

Transdirect Shipping Plugin for Woocommerce gives you access to Australia's largest courier and freight companies, and allows your clients a seamless checkout process.

The main features of this plugin are:

* Free to use.
* Simple configuration.
* Great shipping rates from Toll Ipec, Fastway, Couriers Please, Toll Priority, Northline, Mainfreight and more.
* Seamless checkout process for your clients.
* Multiple quote options.
* Insurance options available.
* Box Sizing
* Order Sync Functionality

Requirements: 

* PHP version at least 5.3.

== Installation ==

This section describes how to install the plugin and get it working.

= Manual installation =

1. Download and install plugin WooCommerce - excelling eCommerce. See Wordpress instructions on [how to do this](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
2. Download and install this plugin, Transdirect Shipping. See Wordpress instructions on [how to do this](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
3. Under Woocommerce settings, navigate shipping tab then Transdirect Shipping option.
4. For Authentication, click Enable Transdirect to enable the plugin. Provide Title Name of this plugin. Email and Password will be provided by Transdirect, please contact sales@transdirect.com.au.
5. For Warehouse Address, enter your delivery postcode.
6. For Default Item Size, if the product size is not set on Woocommerce Product option, this will be used for quote calculation.
7. For Display Options, fill in values that require in your quote.
